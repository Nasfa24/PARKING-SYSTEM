Parkir Pintar 🅿️

Aplikasi Kasir Parkir Tanpa Biaya Server

Langkah 1: Buat Database

Klik tombol di bawah untuk menyalin Master Database ke akun Google milikmu sendiri.

📋 Salin Database ke Google

Langkah 2: Aktifkan Sistem

Di Spreadsheet yang baru terbuka, klik menu Ekstensi > Apps Script.

Lalu di pojok kanan atas, klik Terapkan (Deploy) > Deployment Baru. Pilih akses ke "Siapa Saja" dan salin URL Web App yang muncul.

Langkah 3: Hubungkan

Masuk ke menu Pengaturan di aplikasi ini, dan tempelkan (paste) URL Web App yang tadi kamu dapatkan beserta tarif parkirmu.
